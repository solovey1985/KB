## Study Flow

1. Start with joins, grouping, CTEs, and advanced query patterns.
2. Read data modification and transactions before practicing writes.
3. Study stored procedures as a standalone topic after basic T-SQL syntax is comfortable.
4. Use the stored procedures concept and interview pages for recall practice.
