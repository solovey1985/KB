## Study Flow

1. Start with [General SQL](../general/index.md) for vendor-neutral foundations.
2. Use the [T-SQL Study Index](t-sql/index.md) to move through SQL Server language features.
3. Study [Stored Procedures](t-sql/stored-procedures.md) as a standalone programmability topic.
4. Use the [Interview Practice](sql-server-querying-and-indexing.interview.md) page to focus on SQL Server and T-SQL distinctions.
5. Read the [Concept Map](sql-server-querying-and-indexing.concept.md) and then go deeper into the T-SQL notes.
