# TypeScript Study Index

This section covers TypeScript as a dedicated topic, from fundamental type system concepts through reusable type design and advanced type manipulation.

## Levels

1. [Basic](basic/index.md)
2. [Intermediate](intermediate/index.md)

## Suggested Study Flow

1. Start with [Basic](basic/index.md) for what TypeScript is, basic types, inference, variables, interfaces, and enums.
2. Continue with [Intermediate](intermediate/index.md) for functions, classes, access modifiers, inheritance, and common object typing patterns.
3. Finish with [Advanced](advanced/index.md) for generics, utility types, mapped and conditional types, and stronger project-level compiler thinking.
