# CSS Study Index

This section organizes CSS from foundational syntax through layout systems, interaction styling, performance, and SCSS.

4. [UI States, Forms, and Interaction](ui-interaction/index.md)

## Suggested Study Flow

1. Start with [Foundations](foundations/index.md) for selectors, cascade, specificity, and resets.
2. Continue with [Box Model and Layout Basics](box-model-layout/index.md) for sizing, display, positioning, and centering.
3. Move to [Modern Layout and Responsive Design](layout-responsive/index.md) for Flexbox, Grid, and media queries.
4. Add [UI States, Forms, and Interaction](ui-interaction/index.md) for pseudo-classes, pseudo-elements, transitions, and form styling.
5. Finish CSS proper with [Advanced CSS Engineering](advanced-engineering/index.md).
6. Then study [SCSS](scss/index.md) as a preprocessor layer on top of CSS.
