## Suggested Study Flow

1. Start with [Document Foundations](foundations/index.md) for document structure, metadata, and basic element behavior.
2. Continue with [Semantic Layout and Content Meaning](semantic-layout/index.md) for layout tags and meaningful content structure.
3. Move to [Forms and Validation](forms-validation/index.md) for inputs, native validation, and submission concepts.
4. Add [Accessibility and Inclusive HTML](accessibility/index.md) to connect markup to real user needs.
5. Use [Media and Responsive Content](media-responsive/index.md) for images, audio, video, and embedded content.
6. Finish with [Advanced HTML Patterns](advanced-patterns/index.md) for `data-*`, script loading, templates, metadata, and resilient markup.
