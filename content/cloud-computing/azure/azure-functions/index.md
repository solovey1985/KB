# Azure Functions

## Overview
Azure Functions is a serverless compute service that lets you run event-driven code without provisioning or managing infrastructure. Choose when to focus on business logic while Azure handles the scaling and management.

## Key Concepts
- **Serverless Model**: Pay only for what you use
- **Event-Driven**: Triggers from various Azure services
- **Bindings**: Declarative connections to data and services
- **Stateless Execution**: Each invocation is independent
- **Auto-Scaling**: Automatically scale based on demand
