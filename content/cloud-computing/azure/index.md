# Microsoft Azure

## Overview
Azure is Microsoft's comprehensive cloud platform offering infrastructure, platform, and software services. This section focuses on Azure services commonly used in enterprise applications.

## Services

### [Azure Functions](azure-functions/)
Serverless computing for event-driven workloads. Execute code without managing servers.

- Triggers & Bindings
- Durable Functions
- Performance & Scaling
- .NET Integration
