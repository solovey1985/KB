Cloud computing platforms provide scalable infrastructure and managed services for building modern applications. This section covers major cloud providers and their core services.

## Cloud Providers

### [Azure](azure/)
Microsoft Azure services for building enterprise applications, from serverless functions to full platform services.

### [AWS](aws/)
Amazon Web Services - comprehensive cloud platform (coming soon).

### [GCP](gcp/)
Google Cloud Platform - data and analytics focused cloud services (coming soon).
