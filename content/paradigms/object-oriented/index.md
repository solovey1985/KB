# Object-Oriented Programming Study Index

This section covers object-oriented programming from core vocabulary and class design into relationships, SOLID principles, patterns, and advanced trade-offs.

## Levels

1. [Basic](basic/index.md)
2. [Intermediate](intermediate/index.md)
3. [Advanced](advanced/index.md)

## Suggested Study Flow

1. Start with [Basic](basic/index.md) to learn the core language of objects, classes, encapsulation, inheritance, and polymorphism.
2. Continue with [Intermediate](intermediate/index.md) to understand composition, abstractions, and SOLID-driven design choices.
3. Finish with [Advanced](advanced/index.md) for patterns, anti-patterns, runtime trade-offs, and maintainable large-system design.

## Coverage

- object-oriented fundamentals
- class and object modeling
- encapsulation, abstraction, inheritance, and polymorphism
- cohesion and coupling
- composition and object relationships
- interfaces, abstract classes, and substitutability
- SOLID principles
- design patterns and anti-patterns
- advanced design trade-offs
