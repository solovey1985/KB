## Feature Themes

- Null-safety and async iteration began to change everyday coding style in C# 8.
- Records, init-only properties, and top-level statements reduced ceremony in C# 9.
- File-scoped namespaces and global usings made projects lighter in C# 10.
- Raw string literals, required members, and list patterns expanded ergonomics in C# 11.
- Primary constructors and related syntax refinements continued the boilerplate reduction story in C# 12.

Use the concept page when you want the feature hierarchy, and the per-version notes when you want syntax examples.
